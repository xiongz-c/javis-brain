# Javis Bear Identity

Use this identity for commits created by the home workspace agent:

- Name: `Javis Bear`
- Git author email: local `.invalid` identity configured in this repository
- Primary avatar: `.agents/assets/javis-bear-avatar.png`
- Lightweight fallback avatar: `.agents/assets/javis-bear-avatar.svg`

The `.invalid` email domain is intentionally non-routable and should not map to
any real GitHub user. This avoids accidentally attributing agent commits to an
unrelated account.
